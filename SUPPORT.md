# Support

## Getting Help

### GitHub Issues

Use the GitHub Issues tracker for:

- Bug reports — use the Bug Report template
- Feature requests — use the Feature Request template
- Documentation improvements — use the Documentation template
- Configuration questions — use the Question template

### Before Opening an Issue

1. Search existing issues to see if your question has already been answered
2. Check the README and manual guides in the `manual-guides/` directory
3. Ensure you are running the latest version of the installer
4. Include relevant logs from `/var/log/hydrodactyl-*.log` if reporting a bug
